# Prompt — 4 opciones de icono de app (HundeZonen)

Una sola imagen con **4 opciones** de icono de lanzador en rejilla 2×2, bien separadas para recortar cada una.
Icono actual: pata blanca + pin de ubicación + ruta serpenteante sobre fondo teal.
Hay que modernizarlo con la línea de los vectores nuevos (trazo teal + filo cálido mostaza), pero como **icono de app** (cuadrado redondeado, sólido, legible muy pequeño en la pantalla del móvil, con ganas de tocarlo).

---

```
Create ONE image containing 4 separate mobile APP LAUNCHER ICONS arranged in a
2x2 grid, evenly spaced with clear empty gaps between them so each icon can be
cropped out individually. Plain neutral light-gray background around the grid.

Each of the 4 is a DISTINCT design option for the SAME app: a dog-walking app
("HundeZonen") about discovering walking routes and dog-friendly zones.

SHARED STYLE (all 4 must look like the same brand family):
- Square icon with large rounded corners (iOS/Android adaptive style), full-bleed solid background
- Brand color: dark teal #1AA39A / #257564 (use as background fill or as the main shape)
- A friendly warm-yellow / mustard accent #E8B84B (a thin offset outline, a pin, or a small highlight)
- Clean, modern, flat, minimal — bold simple silhouette, NO text, NO gradients overload, NO photo realism
- Reads instantly at very small size; centered motif with comfortable padding inside the square
- Soft, rounded, premium and inviting feel — makes you want to tap it

THE CORE MOTIF (present in all 4, the brand is recognizable):
a dog PAW PRINT combined with a LOCATION PIN and/or a winding walking ROUTE/path.

THE 4 OPTIONS (make them clearly different from each other):
1. Solid teal rounded square, a chunky white paw print whose lower part flows into a
   winding white route line ending in a small mustard location pin (modern refresh of the current icon).
2. White/cream rounded square, a bold teal paw print as a location-pin shape (paw inside a teardrop pin),
   thin mustard offset outline, tiny route dots underneath.
3. Teal rounded square, a minimal line-art version: white outlined paw + dotted route + mustard pin,
   thin double-line (teal/mustard) look matching the website feature icons.
4. Gradient teal rounded square (light teal to deep teal), a friendly paw print formed by a location pin
   and rounded toe-beans, a small mustard sparkle, very polished app-store quality.

Output high resolution, sharp edges, each icon roughly 1024x1024 within its grid cell.
```

Si los pega muy juntos: añade *"add wide margins and visible gaps between the 4 icons, do not let them touch"*.
Cuando tengas la hoja, pásamela y recorto las 4 opciones para que elijas.
